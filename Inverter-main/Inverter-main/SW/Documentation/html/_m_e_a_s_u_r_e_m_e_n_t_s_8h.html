void calibrate_offsets ( volatile uint16_t rawADC[],
volatile float currentOffsets[],
uint32_t numSamples )

Calibrate the current sensor offsets.

This function calculates the average offset for each current sensor channel by reading the ADC values when no current is flowing. The calculated offsets are used to correct the sensor readings.

Parameters
[in]rawADCBuffer containing the raw ADC values for the channels.
[out]currentOffsetsArray to store the calculated offsets for each current channel.
[in]numSamplesNumber of samples to average for the offset calculation.
